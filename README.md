# URCDE — Universal Reality & Civilization Decision Engine | 宇宙规则与文明决策统一引擎

A bilingual conceptual framework that unifies physics + civilization decisions as one continuous recursive system. Twelve layers, side-by-side EN+中文 throughout, with multiple interactive elements.

## 12 Layers

01. Unified Reality Axioms — Physical + Informational + Emergence + Intelligence (4 layers)
02. Civilization as Physical Dynamic System — 6 state variables + evolution equation
03. Intelligence-Driven Decision Layer — the unified decision function
04. **Co-Evolution Loop** — animated SVG of Physics ↔ Intelligence ↔ Decision ↔ Environment
05. Multi-Layer Simulation Space — Physical / Technological / Social / Decision
06. **Multi-Agent Interpretation** — same scenario, 5 reasoning agents (Physics · AI · Economist · Sociopolitical · Risk) interactive tabs
07. Unified Simulation Scenarios — AI Acceleration · Collapse · Singularity · Abundance
08. **Global State Vector Simulator** — 7 sliders that compute civilization regime live
09. **Counterfactual Universe Generator** — 4 alt universes (Different Constants · Earlier Intelligence · Hive Civilization · AI-First Society)
10. Stability & Collapse Theory — conditions, triggers, phase transitions, recovery
11. Emergent Meta-Laws — 4 higher-order regularities
12. **Civilization Performance Metrics Radar** — 5 indices (IDI · EUE · CSS · ERF · DCI) recompute live from §08 sliders

Closes with the ∞ manifesto.

## Honest framing

URCDE is not predictive AI. It is a structured conceptual atlas — a thinking tool. The "self-generating" property is implemented honestly via interactive elements (state-vector sliders, agent tabs, counterfactual generator) that respond to user input.

## Design

- Plain HTML / CSS / JS, no build step
- Strict side-by-side bilingual layout
- Deep ocean palette `#040814` with cyan/orange duality (cool/hot, physics/intelligence) + gold/lime/violet/rose accents
- Cormorant Garamond italic display + Source Serif body + Noto Serif/Sans SC + JetBrains Mono labels
- Animated starfield background + scroll-reveal + animated co-evolution loop SVG with traveling dashes
- Mobile-responsive
